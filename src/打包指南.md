# 产品追踪器 - Android APP 打包指南

## 方案概述

使用 **Capacitor** 将现有的 web 应用打包成原生 Android APP。

Capacitor 是一个跨平台的原生运行时，让 web 应用可以调用原生 API，生成真正的原生应用。

---

## 第一步：在云开发平台部署

由于这是云开发框架项目，需要先部署到云开发平台：

1. 将代码部署到云开发静态托管服务
2. 获取应用的访问 URL（格式如：`https://xxx.cloudbase.net`）

---

## 第二步：创建 Capacitor 项目（推荐方式）

### 方法一：创建独立 Android 项目

```bash
# 创建项目目录
mkdir product-tracker-app
cd product-tracker-app

# 初始化 npm
npm init -y

# 安装 Capacitor
npm install @capacitor/core @capacitor/cli
npm install @capacitor/android
npm install @capacitor/status-bar @capacitor/splash-screen

# 初始化 Capacitor
npx cap init 产品追踪器 com.producttracker.app

# 添加 Android 平台
npx cap add android
```

### 修改电容配置文件

编辑 `capacitor.config.ts`：

```typescript
import { CapacitorConfig } from '@capacitor/cli';

const config: CapacitorConfig = {
  appId: 'com.producttracker.app',
  appName: '产品追踪器',
  webDir: 'www',
  server: {
    // 修改为你的云开发部署 URL
    url: 'https://your-app.cloudbase.net',
    cleartext: true
  },
  android: {
    backgroundColor: '#ffffff'
  }
};

export default config;
```

### 创建 Web 资源目录

```bash
mkdir -p www
```

将你的 web 应用构建产物复制到 `www` 目录。

### 构建并同步

```bash
# 构建 Capacitor 项目
npx cap copy android
npx cap sync android

# 打开 Android Studio
npx cap open android
```

在 Android Studio 中点击 "Build" → "Build APK" 生成安装包。

## 方法二：使用在线打包平台（最简单）

如果不想配置本地开发环境，可以使用以下在线打包服务：

### 1. HBuilderX

1. 下载 [HBuilderX](https://www.dcloud.io/hbuilderx.html)
2. 将 web 项目导入 HBuilderX
3. 右键项目 → "发行" → "原生APP-云打包"
4. 选择 Android 平台，配置证书
5. 等待打包完成，下载 APK

### 2. DCloud

1. 访问 [DCloud 离线打包](https://nativesupport.dcloud.net.cn/AppDocs/package/Android)
2. 按文档配置 Android Studio 项目
3. 将 web 资源放入 `assets` 目录
4. 构建 APK

### 3. APICloud

1. 注册 APICloud 账号
2. 创建应用，上传代码
3. 使用云编译生成 APK

---

## 方法三：使用 WebView 封装（最简单）

### 使用 WebIntoApp.com

1. 访问 [WebIntoApp](https://webintoapp.com/)
2. 输入你的网站 URL
3. 配置应用名称和图标
4. 一键生成 APK

### 使用 AppsGeyser

1. 访问 [AppsGeyser](https://appsgeyser.com/)
2. 选择 "Web App"
3. 输入网站 URL
4. 填写应用名称
5. 点击 "Create" 生成 APK

---

## APK 输出位置

| 打包方式 | APK 位置 |
|---------|---------|
| Capacitor + Android Studio | `android/app/build/outputs/apk/debug/app-debug.apk` |
| HBuilderX | 项目目录下 `dist` 文件夹 |
| AppsGeyser | 下载页面直接提供 |

---

## 安装到手机

1. 将 APK 文件传输到手机
2. 在手机上打开文件管理器
3. 点击 APK 文件安装
4. 如果提示"禁止安装未知来源应用"：
   - 进入 "设置" → "安全"
   - 开启 "未知来源" 或 "允许安装未知应用"

## 注意事项

1. **扫码功能**：Android 需要相机权限，在 Manifest 中添加：
   ```xml
   <uses-permission android:name="android.permission.CAMERA" />
   ```

2. **语音输入**：需要麦克风权限：
   ```xml
   <uses-permission android:name="android.permission.RECORD_AUDIO" />
   ```

3. **网络权限**：确保应用可以访问网络

4. **HTTPS**：Android 9+ 默认禁止明文流量，需要使用 HTTPS 或配置 `cleartext`
