# 测试用例说明

## 测试文件列表

### 1. test-storage.js
测试存储功能模块（lib/storage.js）：
- 产品数据加载、保存、添加、更新、删除
- 用户信息加载、保存、更新
- 边界条件测试

### 2. test-navigation.js
测试导航功能模块（App.jsx）：
- 页面路由切换
- URL 参数解析
- hashchange 事件处理
- 导航参数传递

### 3. test-product-flow.js
测试产品管理完整流程：
- 首页产品列表显示
- 添加产品（手动、扫码、语音）
- 搜索和筛选功能
- 产品详情查看
- 产品编辑和删除

### 4. test-settings.js
测试设置页面功能：
- 用户信息展示
- 用户信息编辑
- 设置项切换
- 设备同步显示

## 运行测试

```bash
# 安装依赖
npm install

# 运行所有测试
npm test

# 运行特定测试文件
npm test -- test-storage.js

# 运行测试并显示覆盖率
npm test -- --coverage
```

## 发现并修复的问题

1. **导航函数调用方式不一致**
   - 原问题：部分页面使用对象形式调用，部分使用直接调用
   - 修复：App.jsx 中的 navigateTo 函数支持两种调用方式

2. **TabBar 添加按钮无功能**
   - 原问题：中心添加按钮没有绑定点击事件
   - 修复：添加 onAddClick 回调支持

3. **数据保存后页面不刷新**
   - 原问题：保存数据后返回首页数据未更新
   - 修复：导航时更新 dataVersion 触发数据重新加载
